## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start <app>

# watch mode
$ yarn start:dev <app>

# production mode
$ yarn start:prod <app>
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
